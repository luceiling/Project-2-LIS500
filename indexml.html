To Use Our Model:

  1. Upload an image
    • Select "Choose File"
    • Use a .JPG or .PNG for best results. File types like .HEIC are not supported in this model
  2. Interpreting the model
    • The model classifies an image as "not real human" or "real human." Please read information on how we trained our model on the Machine Learning Page to see more information on this
    • The number next to the classification is the rate of confidence the model feels on the classification. .99 is the most confident, and it will go down to .5 before switching to the other classification. This means that if something is classifying at .53 confidence, the model is very unsure whether the image is human or not.
